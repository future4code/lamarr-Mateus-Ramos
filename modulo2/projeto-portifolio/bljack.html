BLACKJACK é um jogo de cartas muito famoso em todo mundo. É fonte de entretenimento de diversas e pessoas e, naturalmente, buscamos criar uma versão dele durante o apredizado do uso de Java Script. Assim, utilizando metodos como for, while, dentre outros, criamos uma versão digital do jogo.
